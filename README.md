# snakeGame

console snake game written in cpp using Ncurses library

build using make commend

execute: ./game

snake head symbol: * <br />
snake body shape: *oooooo <br />
snake body shape: *oo#ooo <br />
#: fruit in the stomach <br />
fruit: @ <br />
